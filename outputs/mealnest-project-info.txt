MEALNEST - LANDING PAGE PROJECT

Scopul proiectului
------------------
Reprodu un landing page simplu pentru o aplicatie fictiva numita MealNest.
MealNest ajuta oamenii sa isi planifice mesele pe saptamana, sa organizeze
retete si sa creeze liste de cumparaturi.

Nivel recomandat
----------------
Incepator spre intermediar:
- HTML pentru structura paginii
- CSS pentru layout, culori, responsive design
- putin JavaScript pentru meniu mobil, scroll smooth sau FAQ accordion

Fisiere recomandate
-------------------
index.html
style.css
script.js

Structura paginii
-----------------
1. Navbar
- Logo: MealNest
- Linkuri: Features, Pricing, Reviews
- Buton: Start free

2. Hero section
- Titlu:
  Plan your meals without the weekly chaos
- Text secundar:
  MealNest helps you organize recipes, build shopping lists, and prepare healthier meals in less time.
- Butoane:
  Get started
  Watch demo
- In dreapta: un card de preview al aplicatiei cu:
  This week
  Monday - Chicken rice bowl
  Tuesday - Veggie pasta
  Wednesday - Salmon salad
  Badge sau progress text: 5 meals planned

3. Features section
- Card 1:
  Weekly meal planner
  Plan meals for each day in a clean calendar view.
- Card 2:
  Smart grocery list
  Ingredients are grouped automatically by category.
- Card 3:
  Recipe library
  Save your favorite recipes and reuse them anytime.

4. How it works
- Pasul 1: Add recipes
- Pasul 2: Pick your meals
- Pasul 3: Shop with one list

5. Reviews section
- Testimonial 1:
  "I stopped ordering takeout every other night."
  Ana M.
- Testimonial 2:
  "The grocery list alone saves me 30 minutes weekly."
  David R.

6. Pricing section
- Plan Free:
  5 recipes
  Basic meal planner
  Manual grocery list
  Buton: Start free
- Plan Pro - $8/month:
  Unlimited recipes
  Smart grocery list
  Weekly templates
  Buton: Go Pro

7. Footer
- Logo mic: MealNest
- Text: (c) 2026 MealNest
- Linkuri: Privacy, Terms, Contact

Stil vizual
-----------
- Fundal: #fffaf3 sau alb cald
- Culoare principala: #2f7d4f
- Accent: #f59e0b
- Text inchis: #1f2933
- Carduri: alb sau crem foarte deschis
- Border radius: 8px
- Font recomandat: system-ui, Inter sau Poppins

Layout
------
Desktop:
- Navbar pe un rand
- Hero section pe doua coloane
- Features in 3 coloane
- How it works in 3 pasi
- Reviews in 2 coloane
- Pricing in 2 carduri

Mobile:
- Totul pe o coloana
- Meniu hamburger optional
- Cardurile se aseaza unul sub altul
- Butoanele pot fi full width

Idei simple de JavaScript
-------------------------
- Meniu mobil hamburger
- Scroll smooth catre sectiuni
- FAQ accordion, daca adaugi o sectiune FAQ
- Schimbare Monthly / Yearly pentru pricing, optional

Ordine recomandata de lucru
---------------------------
1. Creeaza structura HTML.
2. Stilizeaza navbar-ul si hero section.
3. Adauga cardul de preview din hero.
4. Creeaza sectiunea de features.
5. Adauga how it works, reviews si pricing.
6. Fa pagina responsive.
7. Adauga un mic script JS.

Obiectiv final
--------------
O pagina curata, coerenta si responsive, suficient de buna pentru un prim
proiect de portofoliu frontend.
